ChatGPT workflow

Fastest way to log food

Speak into ChatGPT like normal. At the end of the day say: “Add today’s macros to my food diary site.”

I can then update the repository’s daily-log.json. When GitHub Pages refreshes, your site will show that day in the diary.

Fastest way to save recipes

Say: “Add this recipe to my recipe site.” I’ll update recipes.json and ingredients.json so it becomes searchable and scalable.

Important

The website can save edits in your browser. ChatGPT can update the GitHub files, but it cannot directly write into your phone browser’s local storage. For permanent updates across devices, use the ChatGPT → GitHub workflow.
